Установка: 
- Папку banlist загрузить в корень сайта.
- Настроить подключение к базе данных, отредактировав файл config.php.
- Через Админпанель DLE создать cтатическую страницу с названием "banlist", описанием "Банлист проекта", в пункте "Использовать шаблон" поставить: banlist.tpl.
- Нажать "Сохранить".
- Загрузить файл banlist.tpl в папку с вашим шаблоном.
- импортировать файл banlist.sql в базу данных.
- Банлист будет доступен по ссылке http://вашсайт.ru/banlist.html
